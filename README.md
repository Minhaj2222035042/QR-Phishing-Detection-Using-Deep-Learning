# QR Phishing Detection Using Deep Learning

## Project Overview
This research project proposes a deep learning based approach for detecting phishing QR codes through image analysis.

## Research Objective
To identify malicious QR codes and visual deception attacks using deep learning models.

## Models Used
- CNN
- MobileNetV2
- EfficientNetB0
- ResNet50
- DenseNet121
- VGG16
- InceptionV3

## Dataset
A dataset containing legitimate and phishing QR code images was used for training and evaluation.

## Evaluation Metrics
- Accuracy
- Precision
- Recall
- F1-Score
- Confusion Matrix

## Results
EfficientNetB0, MobileNetV2 and CNN achieved approximately 99% accuracy.
